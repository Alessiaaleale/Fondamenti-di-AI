How run code
============
Clona il repository
```python
git clone https://github.com/Alessiaaleale/Fondamenti-di-AI
```
Crea un ambiente virtuale
```python
python -m venv env
```
Attiva l'ambiente virtuale
```python
.\env\Scripts\activate
```
Installa le dipendenze
```python
pip install -r requirements.txt
```
